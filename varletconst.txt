- What is the difference between ***var*** and ***let***?
    
    You can reassign and redeclare with var, but you can not redeclare using let. Let is a block scope and var is a function scope.
    
- What is the difference between ***var*** and ***const***?
    
    You can reassign and redeclare with var, but you can not redeclare or reassign using const. Const is a block scope and var is a function scope.
    
- What is the difference between ***let*** and ***const***?
    
    You can reassign with let, but you can not redeclare or reassign using const.
    
- What is hoisting?
    
    It’s the way that we explain how the JS compiler works. Variables are lifting or “hoisted” to the top of the scope they are declared in.
